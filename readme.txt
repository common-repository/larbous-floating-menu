== Description ==
This plugin creates a floating menu set to the left position.

You can set:
* Choose a custom menu
* the color of the menu options: font e background
* the vertical position of the screen

Set the options in Settings / Larbous Floating Menu

== Installation ==
Install Instructions

1. Upload the folder larbous-floating-menu to the /wp-content/plugins/ directory of your site
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Upload to media library your icons with size 48x48 or similar
4. Create a customized Menu in Appearence / Menu
5. Create options like \"Option Item \"
6. Save with any name. This plugin assumes as default the name \"Larbous Floating Menu\"
